AUS-PSX Colour correction chip install instructions.

Home page: www.ozemail.com.au/~bradpogo
Email: bradpogo@ozemail.com.au


The colour correcton chip will enable your PAL Playstation to play NTSC / HK games in full colour, using a standard non-multisystem TV through the standard RCA output jacks of the Playstation.

Please Note: The colour correction chip does not defeat the Boot protection, you still need a mod chip.

Which Board?

[PU-7] [Pu-8] [5502]

Colour chip legend

 

General Notes:

Wire 4 is only used for the 5502 model PSX. Cut this wire off if it is not to be installed on a 5502 model PSX.

It is a good idea to cut any exess lenght of wire off, check correct lenghts before final installation.
